GitHub Code Upload Guide

How to upload your code directly to GitHub through the browser

Step-by-Step Instructions

1

Sign in to GitHub

Go to github.com and sign in to your account. If you don't have an account, create one first.

2

Create a New Repository

Click on the "+" icon in the top right corner and select "New repository". Give your repository a name and description.

3

Initialize with README

Select "Initialize this repository with a README" to create an initial file. This makes it easier to add more files later.

4

Open Your Repository

Click on the repository name to open it. You'll see the README.md file you just created.

5

Add New Files

Click the "Add file" button and select "Create new file". Enter the filename with the proper extension (e.g., index.html).

6

Paste Your Code

Copy your HTML/CSS/JS code and paste it into the editor. Add a descriptive commit message at the bottom.

7

Commit Changes

Click the "Commit new file" button to save your file to the repository.

8

Enable GitHub Pages

Go to Settings → Pages → Select "main" branch as source → Save. Your site will be published at username.github.io/repository-name

Example HTML File

<!-- Example HTML file to upload -->
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>My Fashion App</title>
  <style>
    /* Your CSS code here */
  </style>
</head>
<body>
  <div class="container">
    <!-- Your app content -->
  </div>
  <script>
    // Your JavaScript code here
  </script>
</body>
</html>

Pro Tip

Name your main HTML file "index.html" to make it the default page that loads when someone visits your GitHub Pages site.

GitHub Pages URL

Your published website will be available at:
https://your-username.github.io/repository-name
